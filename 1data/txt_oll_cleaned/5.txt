     Сервис (сервисная система) – это совокупность средств, способов и методов предоставления платных услуг по приобретению, эффективному использованию, обеспечению работоспособности, экономичности, дорожной и экологической безопасности автотранспортных средств в течение всего срока их службы.
     Техническую эксплуатацию автомобилей определяют следующие особенности:
     Значительный средний возраст и большой срок службы автомобилей. В России средний возраст автомобилей составляет 9,7 лет. Средняя наработка с начала эксплуатации составляет 100 – 105 тыс. км. Около 39% автомобилей имеют наработки более 10 лет, а 13% – более 15 лет.

     Представлены следующие требования к сайту:
     * неброский и привлекающий дизайн;

     HTML – это аббревиатура, которая расшифровывается как HyperText Markup Language или в переводе на русский язык «Язык Гипертекстовой Разметки».Язык, с помощью которого можно сообщить браузеру информацию о том какие элементы есть на web-странице и какую смысловую нагрузку они несут.
     PHP–Аббревиатура означает Hypertext Preprocessor«Препроцессор Гипертекста». Широко используемый язык сценариев общего назначения с открытым исходным кодом. Преимуществом PHP является предоставление web-разработчикам возможности быстрого создания динамически генерируемых web-страниц.
     CSS– CascadingStyleSheets«Каскадные таблицы стилей». Язык, который отвечает за визуальное представление документов пользователю. Используется создателями web-страниц для задания цветов, шрифтов, расположения и других аспектов представления документа. 
     JavaScript – это язык управления сценариями просмотра гипертекстовых страниц Web на стороне клиента. Основная идея JavaScript состоит в возможности изменения значений атрибутов HTML-контейнеров и свойств среды отображения в процессе просмотра HTML-страницы пользователем. 
     Denver–это программа, которая выполняет роль локального хостинга – только на компьютере пользователя (без выхода в Интернет). Хостинг – это место для сайта, где будут размещаться файлы и папки.  В Денвер входят программы: Apache – cвободный web-сервер, PHP – препроцессор гипертекста, MySQL – открытая база данных. В Denwer все процессы автоматизированы и установка происходит за несколько кликов. Создатели Denwer проделали огромную работу и значительно облегчили жизнь, простым пользователям. 
 
     Сайт должен обеспечивать корректное отображение данных в следующих браузерах:

     - Mozilla Firefox (версия 40.0 и выше); 
     - операционная система Windows 7, Windows 8 и Windows 10; 
     - место на диске 50 Мбайт.
     Требования к аппаратному обеспечению сервера хостинга: 
     - процессор семейства x86_64; 
     - оперативной памяти не менее 1 Gb; 
     - место на диске не менее 5.8 Mb; 
     Требования к программному обеспечению сервера хостинга: 
     - операционная система Windows / Linux; 
     - сервер Apache; 
     - язык PHP не ниже 5-ой версии; 
     - система управления сервером cPanel. 
     Все эти параметры обеспечивают корректное отображение сайта в web-браузере пользователя.

2.4 Разработка меню	
      
     Меню – важная часть web-сайта, которая помогает пользователям перемещаться по страницам сайта. Любое меню представляет собой список ссылок, ведущих на внутренние страницы сайта.
     Для удобства в работе пользователей, я решил разработать вертикальное меню, которое появляется на всех страницах сайта.
     Для того, чтобы осуществить эту идею я воспользовался HTML и CSS.
     Вертикальное меню состоит из девятиосновных пунктов.
     
     На сайте создана страница «Контакт», на которой содержится форма заполнения для отправки сообщения администратору сайта, на электронную почту. Если у пользователя возникли какие-то вопросы или он нашел ошибку на сайте, он может зайти на страницу «Контакт» и отправить сообщение.

     Тестирование представляет собой процесс выполнения программы с целью проверки правильности результатов ее работы. Оно включает в себя преднамеренное конструирование трудных наборов входных данных, создающих наибольшие возможности для отказа программы.Основной метод для обнаружения ошибок в программе – проведение испытаний. Испытание представляет собой процесс выполнения программы с целью проверки правильности результатов ее работы и соответствия заданным характеристикам. Процесс испытаний включает в себя преднамеренное конструирование сложных наборов входных данных, создающих наибольшие возможности для отказа программы.
     Порядок тестирования сайта:
     - проверка гиперссылок, поиск нерабочих ссылок (привести скриншоты маленького размера всех страниц сайта);
     - тестирование верстки (изменение масштаба отображения);
     - тестирование верстки сайта в режиме отключенной графики;
     - тестирование верстки в режиме изменения размера шрифта;
     - тестирование php- скриптов;
     - тестирование javascript;
     - тестирование кроссбраузерности (открыть сайт в 3-х различных браузерах).
     Тестирование проводилось в следующих web браузерах:
     - Google Chrome;
     - Opera;
     - Firefox.
