     Задание на курсовой проект заключается в разработке web-ресурса для предметной области «Новостной портал». На сайте пользователь сможет ознакомится с интересующими его новостями со всего мира, также предусмотрена обратная связь и гостевая книга для коммуникации с администрацией сайта, чтобы указать на ошибки сайта или просто оставить свой отзыв.
     Целью курсового проекта является создание web-сайта «Новостной портал». Пользователь сайта может ознакомиться с актуальными новостями, статьями и различными материалами по категориям. Для выполнения данной цели были сформулированы и решены следующие задачи:
     ­ изучить основные опции существующих сайтов заведений подобного характера;
     ­ обозначить аудиторию сайта;
     ­ разработать оптимальную структуру;

     ­ тестирование web-сайта.
     Объектом исследования является web-ресурс «Новостной портал».
     Предметом исследования являются программные средства, языки и технологии, предназначенные для создания web-сайта.

     1 Анализ предметной области
     
     1.1 Исследование предметной области 
     
     На сегодняшний день интернет-технологии стремительно развиваются, и, заглядывая в будущее можно предсказать перспективы развития Интернета. Каждый день появляются и новые сайты по самым различным тематикам. 
     Развитие Интернета повлекло за собой переход многих СМИ на новый формат. Появились интернет-СМИ, которые, как и печатные, ТВ СМИ руководствовались правилами журналистики. 
     Это привело к тому, что новости в настоящее время распространяются практически мгновенно по всему миру с помощью специализированных сайтов.
     Новостной сайт – это интернет-издание, специализация которого заключается в сборе и выдаче общетематических новостей или новостных материалов на одну тему.
     Виды новостных сайтов:
     ­ узко-региональные. Освещают новости какого-либо региона, к примеру Оренбургского региона или Воронежской области РФ (Мое-онлайн, Оринфо);
     ­ широко-региональные. РФ, Украина, Казахстан и т.д. (Газета.ру, Риа.ру, Лента.ру);
     ­ узко-тематические. Один веб-проект выдает материалы только о культурной жизни или строительстве, рынке недвижимости и т.д. (ТВ-культура, Астроньюс);
     ­ широко-тематические. Освещаются материалы для широкого круга читателей (все указанные широко-региональные, а еще АиФ, Вести.ру и др.).
     По способу наполнения материалами новостники также подразделяются на несколько видов:
     - парсинг. Сайты-новостники наполняются автоматически путем сбора информации с сайтов-доноров;
     ­ пользовательский контент. Веб-проект наполняют посетители;
     ­ модерирование. Новости размещают редакторы. При этом они могут как рерайтить их из источников, писать самостоятельно или получать от аккредитованных СМИ;
     ­ смешанный вид.
     Исходя из совокупности вышеперечисленных факторов, для разработки сайта для курсового проекта мной был сделан выбор в пользу политематического новостного сайта, со способом наполнения контентом по типу модерирования.
     
     1.2 Требования к разработке
     
     Сайт новостей должен иметь:
     - современный и стильный дизайн;
     - приятную для глаза цветовую схему;
     - удобное, простое и понятное меню навигации;
     - корректное отображение всех страниц сайта на разных компьютерах и в разных браузерах;
     - обратную связь;
     Целевой аудиторией сайта являются в основном люди от 15 до 30 лет, интересующиеся событиями, происходящими в мире. Но пользователь сайта может оказаться кем угодно, поэтому информация на сайте должна быть понятна для всех пользователей. 
     Для того, чтобы пользователь смог оставить свои замечания и предложения, на сайте должна быть обратная связь, с помощью которой будет производиться взаимодействие пользователя и администратора сайта. Также для оценки сайта должна быть гостевая книга, чтобы любой пользователь мог оставить свой независимый отзыв о сайте.
     
     1.3 Выбор средств реализации
     
     Для верстки сайта необходимы хорошие средства реализации, которые помогут разработать действующий ресурс и внедрить сложные функциональные решения.
     Основным средством написания программного кода и придания вида послужили язык разметки web-документа HTML и таблицы стилей CSS. 
     Для написания программного кода использовался бесплатный редактор кода VS Code – относительно новый текстовый редактор, выпущенный Microsoft. Он, также, как и Atom, основывается на оболочке Electron (написанной командой Atom), кардинально отличаясь реализацией самого редактора [6].
     HTML – стандартный язык разметки документов в Интернете. Он был разработан британским учёным Тимом Бернерсом-ли в 1989-1991 годах и создавался как язык обмена научной и технической документацией [1].
     Гипертекстовые документы интерпретируются браузером клиента в удобное для них представление, которое позволяет просматривать различного вида информацию (текст, графика). С помощью средств форматирования HTML можно составить сложную иерархическую структуру из HTML-страниц [2].
     CSS – это формальный язык, служащий для описания оформления внешнего вида документа, созданного с использованием языка разметки [4].
     PHP (Hypertext Preprocessor (Препроцессор Гипертекста)) – это широко используемый язык сценариев общего назначения с открытым исходным кодом. Говоря проще, PHP это язык программирования, специально разработанный для написания web-приложений (сценариев), исполняющихся на web-сервере [5].
     Open Server Panel — это портативная серверная платформа и программная среда, созданная специально для web-разработчиков с учётом их рекомендаций и пожеланий. Программный комплекс имеет богатый набор серверного программного обеспечения, удобный, многофункциональный продуманный интерфейс, обладает мощными возможностями по администрированию и настройке компонентов. Платформа широко используется с целью разработки, отладки и тестирования web-проектов, а также для предоставления web-сервисов в локальных сетях [3].
     
     1.4 Требования к программному обеспечению
     
     Сайт должен обеспечивать корректное отображение данных в следующих браузерах: 
     ­ Google Chrome (версия 49.0 и выше); 
     ­ Opera (версия 12.0 и выше); 
     ­ Mozilla Firefox (версия 40.0 и выше); 
     ­ операционная система Windows 7, Windows 8 и Windows 10; 
     ­ место на диске 50 Мбайт.
     Требования к аппаратному обеспечению сервера хостинга: 
     ­ процессор семейства x86_64; 
     ­ оперативной памяти не менее 1 Gb; 
     ­ место на диске не менее 5.8 Mb. 
     Требования к программному обеспечению сервера хостинга: 
     ­ операционная система Windows / Linux; 
     ­ сервер Apache; 
     ­ язык PHP не ниже 5-ой версии; 
     ­ система управления сервером cPanel.
     Соблюдение данных требований обеспечит работоспособность сайта.

     3 Тестирование сайта
     
     Перед тем, как сдать сайт заказчику, необходимо проверить его на предмет ошибок и недоработок, то есть провести тестирование.
      
     3.1 Общие понятия тестирования
     
     Тестирование сайта является одним из важных этапов разработки. Тестирование и устранение ошибок предшествуют запуску проекта и выполняются после всех остальных этапов, ведь досконально проверять имеет смысл только уже готовый проект. Но иногда бывает уместным протестировать отдельный этап, например, после верстки дизайна нужно просмотреть, как выглядят страницы на разных браузерах и устройствах. [8]
     Этапы проверки:
     ­ тестирование верстки (изменение масштаба отображения);
     ­ тестирование верстки сайта в режиме отключенной графики;
     ­ тестирование верстки в режиме изменения размера шрифта.
     Тестирование – это отклонение фактического результата от ожидаемого, другими словами – это процесс поиска багов (ошибок) [10].
     Тестирование сайта – один из важных жизненных этапов, после которого, предоставляется заказчику готовый проект без ошибок, с хорошей читабельностью, воспринимаемой легкостью, удобством и надежностью [10].

ЗАКЛЮЧЕНИЕ

      В ходе проведения работы были выделены основные задачи сайта, важные особенности целевой аудитории.
     Оценивая проделанную работу можно сделать выводы:
     ­ сайт представляет пользователям все самые актуальные новости со всего мира;
     ­ дизайн сайта соответствует предполагаемым предпочтениям пользователей, времени и целям нахождения потенциальных клиентов;
     ­ реализована навигация с помощью меню по страницам сайта;
     ­ ссылки легко отличимые от основного текста, находятся в одном месте.
     Результатом проделанной работы стал готовый к работе сайт, содержащий 41 страницу. Сайт содержит такие страницы как: «Главная», «Участники», «Контакты», «Гостевая книга», «Обратная связь». Сайт позволяет ознакомиться с новостями по категориям: «Культура», «В мире», «Бизнес», «Спорт», «Политика», «Наука», «Природа», «Экономик», «Шоу-бизнес», «Кино», «Игры». Сайт позволяет воспользоваться: гостевой книгой, чтобы оставить комментарий, обратной связью, чтобы связаться с администрацией.
