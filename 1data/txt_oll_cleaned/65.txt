Введение


	Тема выпускной квалификационной работы   —   разработка web-сайта для ресторана.
     Web-сайт - это интерактивная электронная платформа, доступная через Интернет, которая содержит различные веб-страницы, объединенные общей темой, структурой или назначением. Это цифровое пространство, где пользователи могут получать информацию, обмениваться данными, взаимодействовать с контентом или выполнять определенные действия.
     Основными компонентами веб-сайта являются веб-страницы, которые могут содержать текст, изображения, видео, аудио, анимацию и другие мультимедийные элементы.
     Цель выпускной квалификационной работы является разработка web-сайта для ресторана. Для достижения данной цели были поставлены следующие задачи:
 изучить различные источники о создании веб-сайтов для ресторанов, включая дизайн, функциональность, и пользовательский опыт;
 изучить информацию о графическом оформлении веб-сайтов для ресторанов, включая выбор цветовой схемы, типографии и использование изображений;
 структурировать найденную информацию;
 разработать веб-сайт для ресторана;
 сделать экономическое обоснование работы;
 рассмотреть вопросы информационной безопасности и охраны труда.
     В процессе разработки web-сайта для ресторана были использованы
следующие языки программирования и технологии: HTML, CSS, JavaScript, PHP.
     Разработанный web-сайт для ресторана должен предоставлять информацию о меню ресторана, местоположении и других аспектах в удобном для посетителей виде.
     Работа состоит из введения, четырех основных разделов, заключения, перечня информационных ресурсов из 15 источников.
     Во введении представлены актуальность разработки, основная цель, задачи.
     В первом разделе рассмотрены основные теоретические сведения. Во втором разделе приведены результаты анализа предметной области, связанной с темой выпускной квалификационной работы, сформулированы условия разработки веб-сайта и требования к программному обеспечению.
     В третьем разделе приведено экономическое обоснование работы.
     В четвертом разделе раскрыты вопросы информационной безопасности и
охраны труда.
     В заключении кратко описаны основные выводы по проделанной работе.

    	1.3 Выбор средств реализации
    
    
	HTML (язык гипертекстовой разметки) - это стандартизированный язык разметки, используемый для создания веб-страниц. Он состоит из набора тегов и атрибутов, которые определяют структуру и содержимое веб-документа. HTML используется для описания содержимого веб-страницы, такого как тексты, изображения, ссылки и другие мультимедийные элементы, а также для управления отображением и форматированием веб-страницы в браузере. HTML позволяет создавать интерактивные и многофункциональные веб-страницы, которые можно просматривать на разных устройствах и в разных браузерах.
	CSS (каскадные таблицы стилей) - это язык таблиц стилей, используемый для проектирования и стилизации веб-страниц. Он предоставляет возможность задавать внешний вид и расположение элементов HTML, включая цвета, шрифты, отступы, границы и другие атрибуты. CSS позволяет создавать эстетически привлекательные и профессионально выглядящие веб-страницы, придавая им красивый и современный дизайн. Язык также поддерживает каскадные механизмы, которые позволяют устанавливать приоритеты стилей и наследовать их, позволяя гибко и эффективно разрабатывать стили веб-страниц.
	JavaScript - это интерпретируемый язык программирования, который применяется в веб-разработке для создания интерактивных элементов и динамического содержимого на веб-страницах. Он позволяет добавлять функциональность к веб-сайтам, делая их более динамичными и интерактивными для пользователей. JavaScript используется для создания слайдеров, анимаций, форм валидации, обработки событий, взаимодействия с элементами DOM (Document Object Model) и многого другого. Этот язык программирования выполняется в браузере пользователя и может взаимодействовать с HTML и CSS, что делает его мощным инструментом для создания веб-приложений и динамических интерфейсов.
	PHP - это скриптовый язык программирования, который широко используется для разработки динамических веб-сайтов и веб-приложений. PHP выполняется на сервере, что позволяет ему взаимодействовать с базами данных, обрабатывать данные форм, генерировать динамический контент и управлять сеансами пользователей. Он поддерживает различные базы данных, включая MySQL, PostgreSQL и другие, что делает его мощным инструментом для создания интерактивных веб-приложений. PHP также интегрируется с HTML и другими языками разметки, что обеспечивает удобство и гибкость при разработке веб-приложений.
	Sublime Text 3 - это текстовый редактор, ориентированный на программистов и веб-разработчиков, который предлагает широкий набор функций и возможностей для написания и редактирования кода. Он поддерживает множество языков программирования и разметки, включая JavaScript, Python, HTML, CSS и многие другие. Среди его основных возможностей - быстрая навигация по файлам и строкам, подсветка синтаксиса, автодополнение кода, множественные курсоры для одновременного редактирования, интеграция с системами контроля версий (VCS) такими как Git, SVN и Mercurial, а также мощные инструменты поиска и замены текста. Благодаря своей гибкости и расширяемости с помощью плагинов, Sublime Text 3 является популярным выбором среди разработчиков для повседневной работы с кодом.
	Яндекс.Браузер - это веб-браузер, разработанный компанией Яндекс. Он обеспечивает быстрый доступ к интернету и включает в себя ряд дополнительных функций. Среди его возможностей - высокая скорость загрузки веб-страниц, встроенная защита от вредоносных сайтов, интеграция с сервисами Яндекса (например, Яндекс.Поиск, Яндекс.Маркет), персонализация интерфейса и функционала, поддержка мультимедийных возможностей, оптимизированный режим турбо для медленного интернет-соединения, удобное управление вкладками и многое другое. Эти характеристики делают Яндекс.Браузер популярным выбором среди пользователей, и он часто используется для просмотра веб-содержимого и работы с онлайн-сервисами.

    2  Практическая реализация
    
    
    2.1  Структура web-сайта для ресторана
     
     
     Задача состоит в создании тематического web-сайта с использованием гипертекстового языка разметки HTML, каскадных листов стилей CSS, языка программирования JavaScript. Разработанный сайт должен осуществлять переход по гиперссылкам от одной странице к другой.
     На главной странице присутствуют следующие разделы:
 «Главная» содержание сайта;
 «Каталог» предоставляет информацию о продукции;
 «Доставка» содержит информацию о видах доставки продукции, включая стоимость;
 «О нас» представляет краткую информацию о ресторане;
 «Контакты» содержит контактную информацию, такую как адреса и телефоны.
   	Основная цель сайта для ресторана - предоставить посетителям информацию о заведении, его меню, атмосфере и услугах, привлечь новых клиентов, обеспечить удобство заказа блюд, а также поддерживать взаимодействие с постоянными клиентами через обновления меню, акции и специальные предложения.
   
   
    2.2  Макет web-сайта для ресторана

После этого происходит верстка и программирование, при которых создается структура и функциональность сайта. Обычно начальный этап заключается в работе дизайнера над внешним видом и макетом сайта, после чего эти данные передаются разработчику для дальнейшей реализации.
     Макет сайта - это визуальное представление организации содержимого и элементов на веб-странице или веб-сайте. Он определяет расположение текста, изображений, кнопок и других компонентов, обеспечивая представление о внешнем виде и структуре сайта перед его созданием и разработкой.

     Термин «интерфейс» обычно относится к точке взаимодействия между человеком и устройством, программой или системой. В информационных технологиях интерфейс определяет способ, с помощью которого пользователь взаимодействует с программным обеспечением или устройством. Это может включать в себя элементы пользовательского интерфейса, такие как окна, кнопки, меню, а также методы ввода и вывода данных, такие как клавиатура, мышь, сенсорный экран и голосовые команды. Интерфейс должен быть удобным, понятным и эффективным, чтобы обеспечить удовлетворительный опыт использования для пользователя.
     Интерфейс - это способ, с помощью которого человек взаимодействует с компьютером или другим устройством. Это то, через что нажимают кнопки, прокручиваете экраны или вводите текст для выполнения нужных действий.
     Ключевым является пользовательский интерфейс, который позволяет людям взаимодействовать с компьютером. Успех нового программного продукта на рынке в значительной степени зависит от удобства этого интерфейса.
     Пользовательский интерфейс может быть символьным и графическим.
     Символьный интерфейс - это способ взаимодействия с компьютером или программой, где информация отображается в текстовом виде с использованием символов, а не графических элементов. Этот тип интерфейса часто используется в командной строке операционных систем или в текстовых редакторах, где пользователь вводит команды и получает текстовые ответы. Символьный интерфейс является одним из самых старых и простых типов интерфейсов, но он все еще используется в различных ситуациях, особенно в случаях, когда необходима быстрая обработка или автоматизация задач.
     Графический пользовательский интерфейс - это метод взаимодействия с компьютером, при котором пользователь взаимодействует с графическими элементами, такими как кнопки, меню, окна и иконки, с помощью мыши или сенсорного экрана. GUI предоставляет более интуитивно понятный и удобный способ управления компьютером, поскольку пользователи могут видеть и манипулировать объектами на экране визуально. Этот тип интерфейса широко распространен в современных операционных системах и приложениях, делая использование компьютера более доступным для широкого круга пользователей.
     Элементы интерфейса в GUI часто созданы с использованием метафор, которые отражают их функциональность и свойства. Это способствует легкому пониманию и использованию электронных устройств даже для пользователей без специальной подготовки.

     3.1 Планирование работы
     
     
     Project Libre - это открытое программное обеспечение для управления проектами и планирования задач. Он предоставляет инструменты для создания графиков проекта, распределения ресурсов, отслеживания выполнения задач и управления бюджетом. Project Libre является альтернативой коммерческим программам для управления проектами, таким как Microsoft Project, и широко используется в различных отраслях для управления проектами любого размера и сложности.
     Ресурсы проекта - это все элементы, которые могут быть использованы для выполнения задач и достижения целей проекта. Они могут включать в себя людей, оборудование, материалы, финансовые средства и другие ресурсы, необходимые для успешной реализации проекта. Управление ресурсами проекта включает в себя планирование, выделение, использование и контроль всех доступных ресурсов для оптимизации их эффективного использования в рамках проекта.
     Сетевой график - это метод визуализации последовательности и зависимостей между задачами в проекте. Он представляет собой графическое представление всех задач проекта и их взаимосвязей, позволяя увидеть временные зависимости и определить критический путь проекта. Сетевой график обычно строится с использованием диаграммы Ганта или диаграммы Перт, что помогает планировать и контролировать выполнение задач и достижение целей проекта.
     Диаграмма Ганта - это графическое представление расписания задач в проекте в виде столбцов, отображающих длительность выполнения каждой задачи, и горизонтальных линий, представляющих временные интервалы. Каждая задача представлена отдельной полосой на диаграмме, причем длина полосы соответствует продолжительности задачи. Диаграмма Ганта позволяет наглядно представить хронологию выполнения задач, их последовательность и зависимости, а также определить критический путь проекта. 4 Безопасность и экологичность работы


     4.1	Информационная безопасность


     Информация изначально является набором данных, которые могут передаваться разнообразными способами: устно, письменно, через сигналы и другие средства коммуникации между людьми, автоматами и в природе. В научном понимании информация обладает универсальными свойствами, включающими в себя передачу сведений, сигналов и признаков между субъектами. С философской точки зрения, информация существует независимо от человека и рассматривается как свойство материи.
     В контексте информационной безопасности, этот термин означает состояние защищенности национальных, общественных и личных интересов в информационной сфере. Защита информации включает в себя комплекс мероприятий, которые охватывают правовые, организационные и технические аспекты. Эти меры направлены на идентификацию, предотвращение и минимизацию угроз информационной безопасности и их воздействия на информационные системы.
     Субъекты информационных отношений определяются как владельцы и пользователи информации, а также элементы инфраструктуры, поддерживающие информационные процессы, включая помещения, системы энергоснабжения и связи, а также персонал, работающий с данными. Угроза информации рассматривается как потенциальная возможность несанкционированного воздействия на защищаемые объекты, что может привести к утечке, искажению или разглашению конфиденциальной информации.
     
     
     
     
     4.2	Охрана труда


     Кроме информационной безопасности, важно также обеспечить безопасность труда для разработчиков. В этом контексте рекомендуется проведение обучений и тренингов по охране труда. Разработчики должны быть обучены правильной работе с компьютером и другими офисными устройствами, а также способам предотвращения травм и профессиональных заболеваний.
     Рабочие места разработчиков должны быть эргономично оборудованы, чтобы минимизировать риски возникновения напряжения мышц, болей в спине и других здоровьесберегающих проблем. Кроме того, разработчики должны регулярно делать перерывы в работе для предотвращения перенапряжения и усталости.
     Использование специально разработанного эргономичного оборудования, такого как клавиатуры и мыши, способствует снижению риска возникновения травм рук и запястий. Также, в офисе должны быть установлены средства предупреждения пожаров и аварий, разработаны планы эвакуации и действий в случае чрезвычайных ситуаций.
     Соблюдение нормативов и требований охраны труда, установленных законодательством и политикой компании, является обязательным. Обеспечение безопасности труда для разработчиков веб-приложения не только предотвращает травмы и болезни, но также способствует повышению производительности, улучшению качества работы и общего благополучия персонала.
     
      

     4.3	Мероприятия по улучшению условий труда 
     
     
     Опасные производственные факторы представляют собой элементы рабочей среды, чье воздействие в определенных условиях может вызвать травмы или другие внезапные проблемы со здоровьем. С другой стороны, вредные производственные факторы могут привести к заболеваниям или снижению работоспособности. Для обеспечения комфортной и безопасной среды труда принимаются различные меры, включая технические, технологические, санитарно-гигиенические, организационные и эстетические.
     Работа с компьютером требует соблюдения определенных параметров, чтобы минимизировать нагрузку на глаза и нервную систему. Это включает в себя обеспечение оптимальных значений визуальных параметров и установленных перерывов. Рабочая мебель должна быть настроена таким образом, чтобы соответствовать росту и обеспечивать удобную позу работника. Рабочий стол должен быть правильно сконструирован, чтобы разместить оборудование и обеспечить удобство в работе.
     При работе с электроустановками важно соблюдать организационные и технические меры безопасности. Это включает в себя периодический медицинский контроль работников и использование защитных механизмов, таких как защитное заземление и зануление. Эти меры предназначены для минимизации рисков и обеспечения безопасного рабочего процесса.
     Кроме того, для обеспечения безопасности при работе с электроустановками, необходимо строго соблюдать правила устройства электроустановок, включая правила заземления. Заземление является важной мерой защиты при работе с оборудованием, подключенным к трехфазным сетям напряжением до 1000 В с изолированной нейтралью. Оно способствует снижению потенциалов корпусов оборудования и обеспечивает срабатывание максимальной токовой защиты при возникновении замыкания на корпус.

Заключение


     Итогом выполнения выпускной квалификационной работы является разработанный  веб-сайт для ресторана. 
     В рамках выпускной квалификационной работы были проанализированы источники по созданию веб-сайтов для ресторанов. Можно заметить растущую популярность таких ресурсов в современном ресторанном бизнесе. В целом веб-сайты обеспечивают удобный доступ к информации о заведении, его меню, расположении и услугах. Клиентам предоставляется возможность ознакомиться с ассортиментом блюд и услугами, выбрать подходящее время и место для посещения, а также сделать заказ онлайн. Создание подобного веб-сайта может стать эффективным инструментом для привлечения новых клиентов и повышения уровня обслуживания в ресторане.
     Для достижения цели были решены следующие задачи:
 рассмотрена теоретическая информация по теме;
 определена структура и функциональные возможности сайта;
 разработан дизайн и шаблон сайта;
 проведено наполнение контентом и тестирование сайта;
 проведено планирование этапов работы и сроков их выполнения;
 проведен экономический расчет;
 рассмотрены вопросы безопасности и экологичности работы сайта.
     Web-сайт для ресторана содержит 15  страниц. Web-сайт позволяет пользователю ознакомиться с меню ресторана, сделать заказ, связаться с администрацией с помощью формы обратной связи, оставить свой отзыв на странице отзывов.
